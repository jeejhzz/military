연락 채널

  • 🏢 사내 담당: 인사팀 지준희 매니저
  • 🤝 자문: 인포섀어 황미경 팀장

서류 1~3번 작성과 병무청 팩스 제출은 인포섀어가 대행합니다.

편입 요건 요약

  • • 현역: 자연계 석사 이상
  • • 보충역: 자연계 학사 이상
  • • 공통: 회사 연구내용 관련 전공
  • • 대표의 4촌 이내 혈족 불가
  • • 복무기간: 36개월

전문연구요원 신규 편입 — 업무 가이드

프로세스 요약

요건 확인 → 인포섀어 통보 → 대기자 정보 엑셀(29항목) 회신 → 서류 7종 준비 → 인포섀어 → 병무청 팩스 제출 → 심사(1~3일) → 승인 알림톡 수신일 = 복무 시작일 (복무 36개월)

필요 서류 7종

#서류준비 주체
1편입신청서인포섀어 작성
2성실복무약정 근로조건이행서약서인포섀어 작성
34촌이내 혈족확인서인포섀어 작성
4대학원 전공과 수행업무 연관 진술서 (최대한 자세히!)회사 + 대상자
5최종 학위증대상자
6성적증명서 (전공 과목 확인용)대상자
7신분증 앞면 사본대상자

편입 대기자 정보 엑셀 29개 항목 (인포섀어 서식)

작성 후 인포섀어에 회신하면 서류 1~3번을 작성해서 보내줍니다.

주의사항

  • 병무청 심사 포인트: 자연계 전공 여부 + 연구내용 관련 전공 여부
  • 연관 진술서가 심사의 핵심 — 팀 리더와 협의해 구체적으로 작성
  • 승인 알림톡 수신일이 복무 시작일 (입사일 ≠ 복무 시작일)
  • 4대보험 취득신고 완료 후 가입내역 확인서 발급 가능
